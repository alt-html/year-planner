Lucide ISC

  • Feather fork — lightest stroke style, extremely clean
  • Inline SVG — no font files, pixel-perfect at any size
  • 1,703 icons — curated, not bloated
  • Active community (forked Feather when it stalled)
  • ISC license — maximally permissive, no attribution required
  • Smallest set — may lack niche icons
  • No Google/Microsoft brand icons — need inline SVGs or a supplement
  • JS-based rendering (createIcons) — not a CSS font class
Best for: the "Ink & Paper" theme. Thin elegant strokes match the editorial feel. The lightest visual weight of the three.

Phosphor MIT

  • 6 weight variants per icon (thin, light, regular, bold, fill, duotone)
  • 9,072 icons — broadest coverage including many brands
  • CSS font classes — drop-in replacement for Font Awesome workflow
  • MIT license — no freemium upsell, fully open
  • Consistent rounded style — friendly but professional
  • Slightly rounded aesthetic — softer than Lucide/Tabler
  • Web font files are ~300KB (regular weight)
  • Microsoft brand icon limited to Outlook — no Windows logo
Best for: both themes. Weight variants let you use "light" for Ink & Paper and "regular" for Crisp & Clear from the same set.

Tabler MIT

  • 6,050 icons — comprehensive including all needed brands
  • CSS font classes — easiest migration from Font Awesome
  • MIT license — fully open, no upsell on icons themselves
  • Consistent 1.5px stroke — crisp and technical feel
  • Full brand icon coverage (Google, Apple, Microsoft, GitHub, CC)
  • Tabler sells templates/illustrations separately (icons are fully free)
  • Single stroke weight — no thin/light/bold variants
  • Web font is ~600KB
Best for: the "Crisp & Clear" theme. Technical, precise strokes match the IDE/tool aesthetic. Easiest 1:1 migration from Font Awesome.
Note on brand icons: All three sets include GitHub, Apple, and Creative Commons. For Google and Microsoft logos, Phosphor and Tabler have coverage. Lucide intentionally excludes most brand marks — you'd supplement with inline SVGs from Simple Icons (CC0 license, 3,200+ brand marks) or keep Font Awesome Brands as a small supplement.
