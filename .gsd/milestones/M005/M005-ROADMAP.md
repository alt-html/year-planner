# M005: UI/UX Design Research

**Vision:** Research and present 2-3 visually distinct design directions for the Year Planner, each showing fresher colour palettes, more characterful typography, and nav patterns that get out of the way. User selects a direction for implementation.

## Success Criteria

- 3 self-contained HTML mockup pages in `/mockups/`
- Each mockup is visually distinct from the others
- Each addresses: muddy colours, generic typography, cluttered nav, weekend/today differentiation
- Each includes both light and dark mode treatment
- Brief design rationale accompanies each option
