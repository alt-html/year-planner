# M005 Summary: UI/UX Design Research

## Outcome

Produced 3 visually distinct design mockups. User selected a hybrid of Ink & Paper (A) and Nordic Clarity (B) as two switchable themes. The design was iteratively applied to the live application, matching grid headers, tooltips, modals, and floating controls to the mockup specifications.

## Deliverables

- 5 mockup files in `/mockups/` (3 themes + combined preview + icon comparison)
- Two-theme system implemented via CSS custom properties on `<body data-theme>`
- Light/dark mode support for both themes
- Grid, modal, tooltip, and control styling aligned to mockups
