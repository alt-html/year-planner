## v1.4 Requirements

## Out of Scope

| Feature | Reason |
|---------|--------|
| Bootstrap JS bundle | All interactivity is Vue-reactive; adding BS5 JS would reintroduce ~78KB for zero benefit |
| BS5 Offcanvas component | Conflicts with Vue-reactive rail; would require BS5 JS |
| BS5 Sass Utility API | No build step â€” CDN only |
| BS5 Floating labels | Fights the custom planner UX |
| Composition API migration | Separate concern; Vue 3 Options API works and is stable |
| Tooltip reimplementation | Cell text is always visible; tooltips are redundant |
